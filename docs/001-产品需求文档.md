## 1. 产品概述

**eplat-devtools** 是一款面向 eplat 低代码平台开发者的 Chrome 扩展工具。为脚本编辑器提供缩放控制、代码片段快速插入、URL 路径管理等辅助功能，提升平台脚本开发效率。

### 1.1 目标用户

- eplat 平台的前端/后端开发人员
- 使用平台脚本编辑器（Ace Editor）编写业务逻辑的人员

### 1.2 核心价值

| 痛点 | 解决方案 |
|------|---------|
| 脚本编辑器字体太小，无法调节 | 缩放控制 + 快捷键 |
| 常用函数记不住，需反复查文档 | 代码片段库一键插入 |
| 编辑器窗口太小，影响阅读 | 最大化/恢复 |
| 复制的 URL 是绝对路径，需手动截取 | 自动提取相对路径 |
| 查找替换只能手动点击 | 快捷键唤出 |
| geDialog 弹窗场景不支持 | 扩展选择器覆盖 |

## 2. 功能清单

### P0 — 核心功能（阶段 1：基础迁移）

| ID | 功能 | 描述 |
|----|------|------|
| F-001 | 脚本编辑器缩放 | Alt+=/- 缩放、Ctrl+滚轮缩放、工具栏按钮 |
| F-002 | 编辑器最大化 | Alt+Enter 切换最大化/恢复 |
| F-003 | 缩放持久化 | 缩放级别保存到 chrome.storage，刷新后恢复 |
| F-004 | Ace Bridge 注入 | 通过外链脚本注入到页面上下文操控 Ace 实例 |

### P0 — 核心功能（阶段 2：功能增强）

| ID | 功能 | 描述 |
|----|------|------|
| F-005 | URL 相对路径提取 | 复制 eplat URL 时自动提取为相对路径，放入剪贴板管理 |
| F-006 | 快捷键唤出查询框 | 快捷键触发 Ace Editor 的查找/替换 |
| F-007 | geDialog 支持 | 扩展 DOM 选择器覆盖 geDialog / ui-draggable 容器 |

### P1 — 重要功能（阶段 3：代码片段库）

| ID | 功能 | 描述 |
|----|------|------|
| F-008 | 代码片段列表 | Sidepanel 展示分类代码片段，支持搜索/过滤 |
| F-009 | 一键插入 | 点击"插入"按钮，代码插入 Ace 光标位置 + 高亮3秒 |
| F-010 | 代码预览 | 选中片段后右侧实时预览代码 |
| F-011 | 剪贴板管理 | 管理复制提取的 URL 和代码片段历史 |
| F-012 | 自定义模板导入 | 在设置页导入自定义 JSON 模板文件 |
| F-013 | JSON 预览编辑器 | 导入的模板可用 JSON 编辑器预览和编辑 |

### P2 — 可选功能（阶段 4：延后）

| ID | 功能 | 描述 |
|----|------|------|
| F-014 | AI 对话助手 | Sidepanel AI 对话 → 生成代码 → 插入编辑器 |
| F-015 | 平台知识库 | RAG 检索平台文档/脚本规范辅助生成 |

## 3. Sidepanel 信息架构

```
┌──────────────────────────────────────┐
│  eplat-devtools                      │
├──────────┬──────────┬────────────────┤
│ 代码片段  │  剪贴板   │    设置        │
├──────────┴──────────┴────────────────┤
│                                      │
│  [当前 Tab 内容区]                    │
│                                      │
└──────────────────────────────────────┘
```

### Tab 1：代码片段

- 搜索栏：模糊搜索函数名/描述
- 分类折叠列表：按类别（弹窗函数、表单校验、通用工具...）
- 每个片段卡片：函数名 + 简述 + 「插入」按钮
- 右侧/下方代码预览面板

### Tab 2：剪贴板

- URL 历史列表（自动提取的相对路径）
- 一键复制 / 清空
- 代码片段插入历史

### Tab 3：设置

- 缩放默认值配置
- 快捷键自定义
- 模板管理：导入 JSON 文件 → JSON 预览编辑器查看/编辑
- 主题切换（浅色/深色）

## 4. 非功能需求

| 项 | 要求 |
|----|------|
| 语言 | 全部 UI 文案为**简体中文** |
| 性能 | Content script 注入耗时 < 50ms |
| 兼容 | Chrome 120+、Edge 120+ |
| 安全 | API Key 仅存 background，不暴露到前端 |
| 安装 | GitHub Release 下载 `.zip` / `.crx`，支持解压加载 |
| 构建 | GitHub Actions 自动构建 `.zip` + `.crx` |
