## 项目背景

eplat-devtools（原 dz-script-editor-zoom-extension）是一款为 eplat 低代码平台开发的 Chrome 扩展工具，核心功能是增强平台内置 Ace 脚本编辑器的使用体验。

## 关键技术决策

### 1. Page Bridge 必须保留
eplat 平台的 Ace Editor 实例挂在 `window.ace`，content script 在 isolated world 无法直接访问。必须通过外链 `<script>` 注入到 main world。

在 WXT 中对应方案：`defineUnlistedScript` + `injectScript` + `web_accessible_resources`。

### 2. 通信模式
- **content ↔ bridge**：`CustomEvent`（`__dz_script_editor_zoom__` / `__dz_script_editor_resize__`）
- **sidepanel ↔ background ↔ content**：`chrome.runtime.sendMessage` / `chrome.tabs.sendMessage`

### 3. 模板直接插入
用户不需要参数表单——代码片段直接以 `${占位符}` 形式插入 Ace，用户在编辑器中手动替换。

### 4. JSON 编辑器选型
使用 `vanilla-jsoneditor`（~40KB），支持树形/文本双模式，适合模板预览编辑。

### 5. UI 全中文
所有 UI 文案（Tab、按钮、描述、提示）必须使用简体中文。函数名/代码保持英文。

## 目标页面

| URL 模式 | 说明 |
|----------|------|
| `http://*/eplat/*` | eplat 平台所有页面 |
| `/eplat/show-designer/dz/edit/*` | 设计器编辑页 |
| `/eplat/BE/DZ/graphIndex.jsp` | 编辑器 iframe 宿主 |

## DOM 选择器

| 选择器 | 用途 |
|--------|------|
| `.scriptEditor` | 脚本编辑器根容器 |
| `.scriptEditorContainer` | 编辑器外层容器 |
| `.scriptEditorHeader` | 编辑器头部（toolbar 挂载点） |
| `.ace_editor` | Ace 编辑器 DOM |
| `#globalScriptEditor` | 全局脚本编辑器 ID |
| `.geDialog` | 平台自定义弹窗组件（待支持） |
| `.ui-draggable` | 可拖拽弹窗 class（待支持） |

## 存储键

| 键 | 用途 |
|----|------|
| `dz_script_editor_zoom_{host}` | 缩放级别（按域名隔离） |

## 技术栈

- WXT v0.20.x + TypeScript + React 18
- Vite 6（WXT 内置）
- `@wxt-dev/module-react`
- `vanilla-jsoneditor`（JSON 预览编辑器）
- GitHub Actions CI/CD（`.zip` + `.crx`）
