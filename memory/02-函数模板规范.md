## 模板 JSON 格式

```jsonc
{
  "category": "分类名称（中文）",
  "icon": "emoji 图标",
  "items": [
    {
      "id": "唯一标识",
      "name": "函数名()",
      "description": "功能描述（中文）",
      "tags": ["标签1", "标签2"],
      "template": "实际代码模板，使用 ${占位符} 标记待填充部分"
    }
  ]
}
```

## 约定

1. `template` 中的占位符使用 `${中文名}` 格式，如 `${标题}`、`${URL}`
2. 插入时保持占位符原样，用户在编辑器中手动替换
3. 一个 JSON 文件对应一个分类
4. `id` 在全局唯一，格式 `{category}-{function}`
5. `tags` 用于搜索过滤

## 内置模板目录

```
snippets/
├── dialog-functions.json   # 弹窗函数
├── form-validation.json    # 表单校验
└── common-utils.json       # 通用工具
```

## 自定义扩展

用户通过设置页「导入 JSON 模板」上传符合上述格式的文件，存储到 `chrome.storage.local`，与内置模板合并展示。导入后可通过 JSON 编辑器预览和修改。
